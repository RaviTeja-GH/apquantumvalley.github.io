Cloud Computing

Delivering computing services over the internet

What is Cloud Computing?

Cloud computing is the delivery of computing services such as servers, storage, databases, networking and software over the internet instead of using local computers or physical servers.

Types of Cloud Service Models

  • IaaS (Infrastructure as a Service): Provides virtual machines, storage and networking resources.
  • PaaS (Platform as a Service): Provides a platform to develop and deploy applications.
  • SaaS (Software as a Service): Ready-to-use software accessed via browser (Gmail, Google Drive, Office 365).

Popular Cloud Providers

  • Amazon Web Services (AWS)
  • Microsoft Azure
  • Google Cloud Platform (GCP)

Benefits of Cloud Computing

  • Cost saving (no physical hardware needed)
  • Scalability and flexibility
  • Access from anywhere
  • Automatic backup and recovery
  • High availability
