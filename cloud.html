Cloud Computing

Delivering computing services over the internet

What is Cloud Computing?

Cloud computing provides servers, storage, databases and software over the internet instead of physical hardware. Users can access services from anywhere using a browser or application.

Cloud Deployment Models

Public Cloud
Services are available over the internet and shared among multiple users. Example: AWS, Azure, Google Cloud.
Private Cloud
Dedicated cloud environment used by a single organization for higher security.
Hybrid Cloud
Combination of public and private cloud allowing data and applications to move between them.

Cloud Storage Types

Object Storage
Stores files like images, videos and backups. Example: AWS S3.
Block Storage
Used for operating systems and databases. Example: AWS EBS.
File Storage
Shared file system accessed by multiple machines. Example: network file shares.

Example Cloud Services

  • AWS EC2: Virtual servers in cloud
  • AWS S3: Storage service
  • Azure Virtual Machine: Microsoft cloud server
  • Google Compute Engine: Google cloud server

Why Companies Use Cloud

  • Reduced infrastructure cost
  • High uptime and reliability
  • Scalability during traffic spikes
  • Automatic backups
  • Remote access for employees
